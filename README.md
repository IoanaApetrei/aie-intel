# AI Enabler Strategic Intel

Weekly strategic intelligence reports for Cast AI's AI Enabler product.

**Covers:** Competitor activity · GPU & infrastructure market · Agentic coding DX ecosystem

## How it works

- Every Monday 08:00 UTC, an agent collects data from 50+ sources and produces a report
- Report is written to `reports/YYYY-MM-DD.md` and pushed to this repo
- Vercel auto-deploys the Next.js app — latest report is always live within ~30s

## Adding a report manually

Drop a markdown file into `reports/YYYY-MM-DD.md` and push. That's it.

## Local dev

```bash
npm install
npm run dev
```
